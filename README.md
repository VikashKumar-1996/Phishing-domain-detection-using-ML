# Phishing Domain Detection Using ML

## Introduction
- This project focuses on detecting phishing domains using machine learning techniques.
- It aims to improve the accuracy and efficiency of phishing detection.

## Installation
- Clone the repository using `git clone https://github.com/VikashKumar-1996/Phishing-domain-detection-using-ML.git`
- Install the required packages using `pip install -r requirements.txt`

## Usage
- After setting up, run the main script with `python main.py`
- Ensure to input the needed parameters as described in the documentation.

## Model Details
- The model utilizes various algorithms, including Logistic Regression, Decision Trees, and Random Forests for classification.
- Extensive training and validation datasets were employed to verify the model's performance.
